## About

I build high-performance Webflow sites that rank, convert, and scale — and I fix the ones that don't. Pixel-perfect from Figma, optimized for Core Web Vitals, WCAG-aligned, with clean CMS architecture and interactions that feel native.

## What I'm Great At

- Figma → Webflow builds using Client-First, Relume, and component-driven structure
- Technical SEO and AEO — semantic HTML, metadata, Open Graph, internal linking, crawl efficiency
- Performance tuning — Core Web Vitals, image strategy, asset compression, lazy loading, code cleanup
- CMS at scale — blogs, resource hubs, service trees, filtered collections, content migrations
- Interactions and motion — native Webflow first, GSAP where it earns its weight, lightweight custom JS when needed
- Audits and rescue — full-site SEO and performance reviews, redirect mapping, and hands-on fixes

---

## Services

- New builds — marketing sites, landing pages, multi-page builds
- SEO and performance optimization — technical audits, structured data, speed improvements
- CMS setup — content architecture, migrations, redirects, internal linking strategy
- Accessibility — WCAG compliance fixes and ongoing monitoring
- Interaction design — animation polish, scroll-driven effects, micro-interactions
- Ongoing support — design updates, A/B test variants, iterative improvements

---

## How I Work

- Clear scope, fast iterations, Loom walkthroughs
- Client-First naming, 8pt spacing, tidy Style Guide
- Native Webflow first — custom code only when it genuinely improves the outcome
- QA across every breakpoint, compressed assets, lazy loading by default
- Handoff with notes, mini docs, or a short Loom
